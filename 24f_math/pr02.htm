隨堂練習 02

題目敘述與分數比率

  • (40%)撰寫一 function q1，輸入是三個正浮點數 a、b、c，且 a ≤ b ≤ c，代表三角形的三個邊。請你依照海龍公式，計算三角形面積並回傳，若三邊無法形成三角形時，應回傳 -1。
  • (40%)撰寫一 function q2，輸入是三個正浮點數 a、b、c，且 a ≤ b ≤ c，代表三角形的三個邊。請你先求取外接圓半徑 r，再以 a * b * c / (4 *r) 的方式，計算三角形面積並回傳，若三邊無法形成三角形時，應回傳 -1。
  • (20%)在上述題目中，a、b、c 能形成三角形的情況下，計算兩種方法的誤差之絕對值。

注意事項

  • 本練習允許 import math 模組，並建議將三題撰寫在同一檔案。
  • 若海龍公式或外接圓半徑公式不在你的記憶中，請詢問同學或上網搜尋之。
  • 第三題可以不用寫成 function 的形式。
