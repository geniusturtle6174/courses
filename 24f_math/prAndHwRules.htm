• 隨堂練習 demo 規定
    • 時間：當週課堂時間。
    • 地點：原上課教室。
    • 方式：完成練習後通知教師或助教，教師或助教有空時會立即前往查看。
    • 當日未能 demo 者不給分，請假者另行討論。

  • 期中作業 demo 規定
    • 時間：原定課堂時間。課前及課後時間另視情況開放預約。
    • 地點：原上課教室。若有開放課前及課後時段時，地點另定。
    • 方式：教師或助教依事前公布之順序叫號，同學到教師或助教座位旁進行說明。
    • 當日課堂時間未能 demo 者：
      • 能於次一週課堂時間 demo 者，以八折計分。
      • 能於次二週課堂時間 demo 者，以六折計分。
      • 於次二週課堂時間仍未能 demo 者，不予計分，請假者另行討論。
      • 遇有假期或天災等原因停課時，不計算其週次。

  • Demo 時，需解釋程式與回答問題，視狀況給分。

  • 程式碼必須撰寫於副檔名為 py 的純文字檔，並讓 Python 直接執行該文字檔。其他任何方式，包含但不限於開啟 Python 後逐行輸入者，或者使用 Jupyter Notebook 者，教師得不予採計其分數。

  • 除題目允許外，禁止以「import」或其他任何方式載入函式庫。

  • 若各別作業之規則跟上述內容衝突時，原則上依各別作業之規則為準。
