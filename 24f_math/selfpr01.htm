自我練習 01

題目敘述

  • 撰寫一 function is_mul_of_eleven，輸入是一個未滿 10000 的正整數，請你依照「奇數位數的數字和，與偶數位數的數字和的差為 0 或 11 的倍數」的方式，判斷該數是不是 11 的倍數。若是，請回傳 True，否則請回傳 False。
  • 撰寫一 function real_sol_num，輸入是三個浮點數 a、b、c，且 a 不為 0，代表方程式 ax2 + bx + c = 0 中的 a、b、c。請你計算該方程式相異實根的個數，若有兩相異實根應回傳 2，有重根時回傳 1，無實根時回傳 0。
  • 撰寫一 function get_int_grade，輸入是一個非負浮點數，請將該數當作某人在本課程中取得的原始分數，並依照本課程「成績最高給至百分制 99，小數取整時除 [59, 60) 及 [89, 90) 區間為無條件捨去外，其餘為四捨五入」的原則，計算出對應的本課程成績。為求簡便，其中的四捨五入，可以直接使用「round」計算。
