Q1: Reverse and Add

作業資訊

  • 檔名必須為「q1_你的學號.py」，例如 q1_9962816.py

題目敘述

把一個數字與其倒序相加，並反覆進行此操作，則通常會得到一個回文數，例如 64 + 46 = 110，110 + 11 = 121（其中 110 的倒序為 011，去除前導零後即為 11）。然而，也有一些數字例如 196，經過此反覆操作後，到目前(2025/10/28)為止已計算了數億位，但都尚未能得出回文數。

於本題中，要請你撰寫一個函式 rev_n_add，輸入參數是一個正整數 n，若 n 可以經過若干次操作得到小於等於 21 * 10 ** 9 的回文數，請回傳操作次數；若否，請回傳 -1；若 n 本身已是回文數，請回傳 0.1。In this problem, you are asked to write a function rev_n_add that takes a positive integer n as input. If n can become a palindrome less than or equal to 21 × 10 ** 9 after a certain number of operations, return the number of operations required. If it cannot, return -1. If n is already a palindrome, return 0.1.

輸入說明

一個正整數 n，且 n ≤ 2 * 109。

輸出說明

一個整數，代表前述的操作次數、0.1，或 -1。An integer representing the number of the aforementioned operations, 0.1, or -1.

Sample Input

64

Sample Output

2
