課後練習 02

題目敘述與分數比率

  • (50%)口語上的「取個整數」的目標，通常不是指數學上的整數，而是某個整數的倍數，例如「將 123 取個整數」，視目標不同，可以是 125 或 130，也有可能是 120 或者 100 等結果。請撰寫一個 function q1，計算「當 b 為正整數時，將正整數 a 加上一個盡可能小的非負整數，使結果為 b 的倍數；當 b 為負整數時，將正整數 a 減去一個盡可能小的非負整數，使結果為 b 的倍數」的結果，範例如下：
    • a = 123，b = 5 時，函式應回傳 125。
    • a = 123，b = -6 時，函式應回傳 120。
  • (50%)撰寫一 function q2，輸入是四個正浮點數 x、y、d、theta，且 0 ≤ theta < 2π，依次代表當你將一根針丟到座標平面上時，針的其中一端點的 x 座標和 y 座標、針的長度，以及從該端點指向另一端點的向量與 x 軸正向的夾角（以 x 軸正向為 0，逆時針方向為正）。請回傳 2 個浮點數和一個布林值（即 True 或 False），依次代表另一端點的 x 座標和 y 座標，以及此根針是否有穿越任何 y 為整數的水平線。

注意事項

  • 此練習皆須於當週 demo，排隊 demo 請使用此連結。
  • 你必須將所需的參數準備好後，再傳入 function。以教材「自訂函式」第一個範例來說，「a = 3」或者「a = int(input("Input a number:"))」之類的程式碼，必須僅出現在 function 之外。
  • Function 計算的結果，必須先用 return 傳出去以後，才能再用 print 等方式顯示出來。
  • 第二題允許 import math 模組。
  • 如果你無法自己推導出第二題所需之正確公式，請考慮退選此課程。
  • 本練習之 function 名稱可以自訂 iff 你自認已熟悉本練習所需之程式語法。
