• 練習或作業原則上都要 demo，亦即修課同學必須向教師或助教需解釋程式與回答問題，且教師或助教將視同學的回答狀況給分。

  • 課後練習 demo 規定
    • 時間：依個別題目指定當週或隔週課堂時間，並視情況使用課前或課後時間。
    • 地點：原上課時間在原上課教室，課前或課後時間的地點另行公告。
    • 方式：完成練習後通知教師或助教，教師或助教有空時會立即前往查看，或者請你到教師或助教的座位旁。
    • 未能於指定時間內完成 demo 者將有折扣或不收，請假者另行討論。

  • 作業 demo 規定
    • 時間：指定週次之原定課堂時間，並視情況使用課前或課後時間。
    • 地點：原上課時間在原上課教室，課前或課後時間的地點另行公告。
    • 方式：教師或助教依公布之順序叫號，同學到教師或助教座位旁進行說明。
    • 當次未能 demo 者：
      • 能於次一週課堂時間 demo 者，以八折計分。
      • 能於次二週課堂時間 demo 者，以六折計分。
      • 於次二週課堂時間仍未能 demo 者，將受更重之折扣或不收，請假者另行討論。
      • 遇有假期或天災等原因停課時，不計算其週次。

  • 程式碼必須撰寫於副檔名為 py 的純文字檔，並讓 Python 直接執行該文字檔。其他任何方式，包含但不限於開啟 Python 後逐行輸入者，或者使用 Jupyter Notebook 者，教師得不予採計其分數。

  • 除題目允許外，禁止以「import」或其他任何方式載入函式庫。

  • 若遇有數學符號、程式片段或題目敘述等方面之顯示差異，以課程網頁在 Windows 電腦版的 Chrome 或 Edge 瀏覽器最新版開啟時之顯示效果為準。

  • 對於作業以及隔週繳交的課後練習，教師或助教可能會不定時實施「事前批改」，即提前告知你的繳交是否符合規格以及正確性如何。

  • 若各別作業之規則跟上述內容衝突時，依各別作業之規則為準。
