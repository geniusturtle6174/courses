隨堂練習 02

題目敘述與分數比率

  • 撰寫一 function q1，輸入是一個整數，代表年份；若該年為閏年則回傳 True，否則回傳 False（25%）
  • 撰寫一 function q2，輸入是兩個整數，代表年和月；若月份為 3~12，請回傳原本的年和月，否則請回傳年份 -1 以及月份 +12，亦即將每年的一二兩月，視為上一年的 13 和 14 月（25%）
  • 撰寫一 function q3，輸入是三個浮點數，代表三角形的三個邊；回傳三角形面積，若三邊無法形成三角形時，應回傳 -1（25%）
  • 依照 hw01 的範例要求來撰寫以上程式（三題請撰寫在同一檔案，因此你需要把該範例的 function 整體複製兩次），並將檔名命名為「pr02_你的學號.py」，例如 pr02_9962816.py，並上傳到 eeclass 的「隨堂練習二繳交區」（25%）

注意事項

  • 閏年與否的判斷，可直接參考教材程式碼。
  • 本題允許 import math 模組。
  • 最後一題是讓各位能提早熟悉作業繳交方式用的，demo 仍為現場進行。
