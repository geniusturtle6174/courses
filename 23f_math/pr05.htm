注意事項

  • 如果你忘記了 LeetCode 的作答及測試方式，請參考上週的隨堂練習。
  • 是否為「一行程式」的範例如下，其中前者(c1)的產生為非，後者(c2)的產生為是： 
    a = ['A', 'B', 'C']
b = ['X', 'Y', 'Z']

c1 = []
for i in a:
	for j in b:
		c1.append(i + j)

c2 = [i + j for i in a for j in b]
  • 第三題的提示如下，請觀察兩個輸出的結果差異： 
    a = ['A', 'R', 'Y', 'X', 'C', 'R']
b = [ 5,   1,   2,   3,   2,   4]

print(sorted(zip(a, b)))
print(sorted(zip(b, a)))
