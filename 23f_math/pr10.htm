隨堂練習 10

題目敘述與分數比率

  • 爬蟲題：給定噗浪上某一則噗的網址，例如 https://www.plurk.com/p/pgnh5y，請爬取它被發表的時間
  • 畫圖題（二選一）
    • Sierpinski triangle，繪製步驟如下
      1. 繪製一正三角形
      2. 自選三角形內之一點並繪出
      3. 隨機選取一個頂點
      4. 將前兩步的兩的點取中點
      5. 以第四步算出的中點，帶入第二步
      6. 重複步驟二到五，建議至少一千次
    • Barnsley fern，繪製步驟如下
      1. 以原點為起始點並繪出
      2. 以不同機率選取下列指定方式，移動到下一個點並繪出
        • 1%: xi+1 = 0, yi+1 = yi * 0.16
        • 85%: xi+1 = xi * 0.85 + yi * 0.04, yi+1 = xi * (-0.04) + yi * 0.85 + 1.6
        • 7%: xi+1 = xi * 0.2 + yi * (-0.26), yi+1 = xi * 0.23 + yi * 0.22 + 1.6
        • 7%: xi+1 = xi * (-0.15) + yi * 0.28, yi+1 = xi * 0.26 + yi * 0.24 + 0.44
      3. 重複上一步，建議至少一萬次

注意事項

  • 本週配分為僅解出一題得 70 分，兩題都解出則得 100 分。
  • 本週題目允許使用任何 pip 已提供之函式庫。
