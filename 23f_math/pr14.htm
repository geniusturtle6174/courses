題目敘述與分數比率

  • 請下載此音檔，用 Python 讀取之，並進行以下工作：
    • (30%)回答其取樣率，並根據訊號點數計算其時間長度
    • (30%)承上，假設每個取樣點是 16 bit，請據此計算其檔案大小（記得加上 header size） ，並比對是否與實際值相符
    • (40%)任取一小段，計算其音高。建議從第 11400 個取樣點開始，取 1024 個點做為那一小段；並且，你計算出來的音高值應該要在 360 Hz 到 380 Hz 之間

注意事項

  • 本週題目允許 import 任何 Python 內建或 pip 已提供之函式庫。
  • 計算音高時，手動或自動的方法可以單獨或混合使用，但是 demo 時要說明過程。
