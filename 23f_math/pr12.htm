隨堂練習 12

題目敘述與分數比率

  • 請接續上週的隨堂練習，進行以下工作
    • (35%)以時間（十進位數字形式）為 x，網址（十進位數字形式）為 y，擬合出一條直線，並顯示其直線方程之各項係數
    • (35%)將此噗的時間（十進位數字形式，請注意時區）帶入前一小題的方程式，預測其網址（十進位數字形式即可）
    • (10%)計算你預測的網址，跟實際網址的誤差（皆轉換成十進位數字形式計算即可）為多少
    • (10%)嘗試看看，更換取用的資料範圍時，預測結果及誤差會不會改變
    • (10%)將第二小題預測出的網址，改回原始網址格式（提示：可上網搜尋 python int to different base，但你仍然需要看懂內容）

注意事項

  • 本週題目已相當於一個最簡單的 final project 程式，因此在 final project 當中繳交一模一樣的內容上來，可能不一定會拿到報告分數。因此若你想取得分數，則需要再多做一些事情，例如多收集一些資料，或者多做一些實驗等等。
  • 本週題目允許 import 任何 Python 內建或 pip 已提供之函式庫。
