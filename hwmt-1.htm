Q1: Count Twin Prime

作業資訊

  • 檔名必須為「q1_你的學號.py」，例如 q1_9962816.py

題目敘述

如果連續兩個質數的差是 2 以內，則稱這一對質數是孿生質數，例如 (2, 3)、(3, 5)、(5, 7)、(11, 13) 等等。本題要請你撰寫一函式 count_twin_prime，輸入為兩個正整數 a 與 b，輸出 [a, b] 區間內的孿生質數對的數目。

輸入說明

兩個參數，分別是題目敘述的兩個正整數 a 和 b，且 0 ≤ a ≤ b ≤ 1000。

輸出說明

一個整數，代表 [a, b] 區間內的孿生質數對的數目。

Sample Input

3, 30

Sample Output

4

提示與注意事項

  • 測資只有一組。
  • 範例測資的 4 對，是 (3, 5)、(5, 7)、(11, 13) 以及 (17, 19)。
  • 若要參考教材範例，請盡可能的減少修改量，否則可導致你在 demo 時，被問到範例本身相關的問題。

	pass
