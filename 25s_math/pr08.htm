題目敘述與分數比率

  • (70%)以下是一個將實驗重複五次的範例，請將 MLPClassifier 的參數內容進行修改，例如「hidden_layer_sizes=(32, 32)」，再跑另外五次，並將兩個五次的結果用 boxplot 繪製後，手動用視窗上的儲存按鈕，將繪製結果儲存成 png 檔案。 
    import numpy as np
from sklearn.datasets import load_digits
from sklearn.model_selection import train_test_split
from sklearn.neural_network import MLPClassifier
 
dataset = load_digits()
print('Data shapes:', dataset.data.shape, dataset.target.shape)

for _ in range(5):
	X_train, X_test, y_train, y_test = train_test_split(dataset.data, dataset.target)
	model = MLPClassifier(hidden_layer_sizes=(256, 256))
	model.fit(X_train, y_train)
	pred = model.predict(X_test)
	print('Accuracy: {:.2f}%'.format(100*np.mean(pred == y_test)))
