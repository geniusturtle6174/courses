題目敘述與分數比率

  • (10%)註冊 Kaggle 帳號，並在 demo 時登入給教師或助教看。已有帳號者可以登入現有帳號，不必註冊新的。
  • (10%)下載教材中「基本概念與範例」中提到的資料集，並順利執行程式範例。
  • (40%)承上，請接續該程式範例，畫出教材範例中的圖片，分類或回歸擇一即可。
    • 圖片細節如 xlabel 和 legend 等方面請盡可能相似，否則可能會被扣分。
    • 本題接受隔週 demo，但是會打八折。
  • (40%)承上，請根據你上一題選擇的是分類或回歸，並在不改變教材範例中模型參數的的情況下，來驗證教材範例中計算的 accuracy 或 MAE 是否相近。
    • 本題接受隔週 demo，但是會打八折。
  • (Bouns: 5%)請改變模型參數，並找出一組能比教材範例中計算的 accuracy 或 MAE 更佳者。
    • 此加分題隔週不打折，但前一題未取得分數者，加分題亦不給分。
