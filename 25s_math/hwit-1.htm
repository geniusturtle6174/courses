Q1: Words of Fortune

作業資訊

  • 檔名必須為「q1_你的學號.py」，例如 q1_9962816.py

題目敘述


由於肉眼看字實在太麻煩了，但是讓各位辨識圖片中的字又有些困難，所以本題會以多行字串的方式（但內容可能會有一或多行，下同），並且每行都是對齊的格式來提供資訊給你，請你撰寫一個函式 words_of_fortune，來檢查多行字串當中是否存在某特定單字。請注意，本題不允許看橫的（由左到右）和直的（由上到下），但是會有斜的（由左上到右下）。

輸入說明

第一個參數是一個多行字串，每行的內容只有大寫英文字母，每行的字母數相等，且不會有空行。第二個參數是一個單行字串，內容也只有大寫英文字母，是要尋找的目標單字。

輸出說明

若有找到目標單字，請回傳 True（即「return "True"」），否則請回傳 False（即「return "False"」）。

Sample Input

參數一：
SDRDHE
GOFEGN
WFLCUO
UHUHIK
TKNWYO
BKKCHK
NPWPIU
 參數二：FLUNK

Sample Output

"True"

提示與注意事項

  • 測資只有一組。
  • 程式架構範例如下，建議只修改標為 pass 的地方以及「if __name__ == '__main__':」區塊的內容，否則若影響自動批改時，可能會有折扣： 
    def words_of_fortune(multi_line_str, target):
	pass


if __name__ == '__main__':
	multi_line_str = """SDRDHE
GOFEGN
WFLCUO
UHUHIK
TKNWYO
BKKCHK
NPWPIU"""

	print(words_of_fortune(multi_line_str, 'FLUNK'))
