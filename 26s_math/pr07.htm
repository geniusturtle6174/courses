課後練習 07

題目敘述與分數比率

建議依下列先後順序實作。程式碼都寫在同一或兩個檔案內即可，不必分題號：
  • (0%)至 Kaggle 的 Titanic - Machine Learning from Disaster 比賽，下載相關資料集。
  • (0%)讀取其中的 train.csv，並以 Sex 和 Age 欄位為特徵，以及 Survived 欄位為答案，訓練任何一種模型。
    • 自己用逗點分隔每行會有問題，因此建議使用 csv 函式庫來讀取。
    • 請勿手動對 csv 檔案作任何更動。
    • 可以改為挑選答案欄以外的任何特徵，但此題與下一題必須使用相同的特徵。
    • 於本題中，請假設 csv 檔案與 Python 程式碼在同一目錄下。
  • (100%)讀取其中的 test.csv，並以 Sex 和 Age 欄位為特徵，加上前一步訓練出的分類器，計算出預測結果，並修改成 Kaggle 指定的格式，上傳至 Kaggle 看看預測準確度。
    • 可以改為挑選答案欄以外的任何特徵，但此題與上一題必須使用相同的特徵。
    • 其餘注意事項與上一題相同。
  • (自我練習不計分)請再改用其他的欄位（例如 Pclass）組合為特徵，訓練另一個模型並上傳至 Kaggle，看看預測準確度是否有差異。

注意事項

  • 請透過 eeclass 「課後練習 07 繳交區」，將程式碼貼在文字輸入框內繳交。貼上時，請留意縮排之正確性。
    • 如果你發覺自己不論怎樣貼上，每行程式碼都匯往最左邊靠攏，則可同時以貼上及附件方式繳交程式碼。
    • 教師或助教批改時，將抽選部分同學的程式碼實際執行並上傳 Kaggle，若無法順利在 Kaggle 取得執行結果，則本練習不給分。
  • 若教師或助教認為需要請你來 demo 回答問題，則你最遲會在下次實體上課時被告知。
  • 本週允許使用 numpy、scikit-learn，以及任何 Python 內建的函式庫。
