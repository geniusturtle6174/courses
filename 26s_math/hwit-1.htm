題目敘述

在「獵人」這部作品中，有個角色的能力是盜取別人的能力，但是需要在一小時內達成下列三個條件（可以不用依序發生，也不限只能各發生一次，中間也可以發生其他無關的事情，例如對方喝了一口飲料）：
  • 實際看過對方的能力
  • 詢問對方關於能力的問題，並且對方有回答
  • 對方把手按在某本書的封面上

於本題中，會給你一個正整數 q 和 events、intervals 兩個串列：q 的意義是盜取能力需要滿足編號 1 到 q 的條件，串列 events 的長度是 n，內容都是正整數，代表各種事件發生的順序；串列 intervals 的長度是 n - 1，內容都是正整數，其中 intervals[i] 代表 events[i-1] 與 events[i+1] 兩個事件的間隔時間，單位為秒鐘。

In this problem, you are given a positive integer q and two lists, events and intervals: q means the stealing ability must satisfy conditions 1 to q. events has length n and contains positive integers representing the event sequence; intervals has length n - 1 and also contains positive integers. Each element intervals[i] shows the time in seconds between events[i-1] and events[i+1].

請你寫一個函式 can_be_hunted，輸入是上述的三個參數，輸出是該角色在輸入所示的這段時間中，是否有任意時刻中可成功盜取別人的能力，亦即找尋是否有任何一個 i、j (0 ≤ i < j < n-1)的配對，使得 intervals[i-1] 到 intervals[j]（兩邊皆包含）的和小於等於 3600，且 events[i-1] 到 events[j+1]（兩邊皆包含）的所有元素所成的集合，包含集合 {1, 2, 3}。

Please write a function can_be_hunted. The input of the function is the above three papameters, and the output is whether the character can successfully steal someone else’s ability at any moment during the given time. In other words, check if there is any pair of indices i and j (0 ≤ i < j < n-1) such that the sum of intervals[i-1] to intervals[j] (both ends included) is less than or equal to 3600, and the set of all elements from events[i-1] to events[j+1] (both ends included) contains the set {1, 2, 3}.

輸入說明

有三個輸入參數，依次分別代表上述的 q、events 和 intervals，其中 intervals 的長度不為零，即 events 的長度至少為 2。

輸出說明

回傳 True 或 False。True 代表可盜取，False 代表不可盜取。

Sample Input

3, [1, 1, 4, 3, 1, 2, 1, 1], [500, 800, 900, 600, 400, 300, 500]

Sample Output

True
