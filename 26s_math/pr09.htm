課後練習 09

題目敘述與分數比率

請把教材「基於樹的模型」的第二個範例的 make_moons 替換成 make_circles，並依序完成下列題目
  • (20%)手動跑幾次，肉眼觀察大概的 test set accuracy 並回答。
  • (80%)加入點座標與原點的 2-norm 距離（或距離平方，亦即最後要不要開根號都可以），並再度手動跑幾次，重新肉眼觀察大概的 test set accuracy 並回答。

注意事項

  • 本週僅允許該範例有 import 的函式庫。
  • 請透過 eeclass 「課後練習 09 繳交區」，將第二題的程式碼貼在文字輸入框內繳交。貼上時，請留意縮排之正確性。Demo 時，除了可能要回答你觀察到的 accuracies，以及如何加入特徵以外，你可能也需要知道為什麼加了會有效。
  • 提示：若認為 accuracy 差距不明顯，可以將 noise 參數改小一點來執行。
