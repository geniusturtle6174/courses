注意事項

  • 每題 40 分，最高採計至 100 分。
  • 本週有到課並解出至少一題，或者本週內完成正式請假者，可以在次週課堂 demo 未解出之題目並以八折計分，其餘狀況不收遲交。請假者須提供假條截圖，教師亦可能視情況要求你提出其他佐證，例如就醫證明等。
  • Demo 時，教師會要求你回答 SQL 語句的某個部分，是為了符合題目條件的哪個部分；或者題目條件的某個部分，撰寫在 SQL 語句中是哪個部分。對於第三題，教師可能會要求你重新整理，並根據螢幕上的變動來修改指令。
  • 對於第三題，雖然 singer 資料表中的男歌手、女歌手，以及團體都各只有一筆資料，但你的 SQL 指令不應利用此狀況。
