平時練習 03 第二題

陣列內容說明

假設有一個二維陣列，第一層代表許多學生，第二層代表他們的基本資料（固定為僅含姓名和學號），如下： 
$test = array(
	array('name' => 'Alice', 'id' => '411000123'),
	array('name' => 'Bob', 'id' => '411000456'),
	array('name' => 'Cindy', 'id' => '411000789')
);

輸出範例說明

請輸出成為一個表格，如下：
NameID
Alice411000123
Bob411000456
Cindy411000789

注意事項

  • 顯示風格的細節不必完全相同，但是內容和順序要跟原始陣列相同。
