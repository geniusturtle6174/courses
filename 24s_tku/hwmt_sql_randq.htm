期中作業 SQL 區出題

題目內容

  1. 選出帳號(account.acount)為 ，密碼(account.passwd)為 的人的相關資訊。請留意，如果帳號密碼配對不正確，你的 SQL 應該回傳空的結果。
  2. 依照 course 資料表的 欄位的 排序列出課程。
  3. 列出使用者流水號為 的人所參加的所有課程的流水號，依課程流水號遞增排序。
  4. 列出使用者流水號為 的人所參加的所有課程的科號、名稱以及描述，依科號遞增排序。
  5. 列出帳號(account.acount)為 的人所參加的所有課程的流水號及參加身分。

注意事項

  • 選出的欄位可以多，但是不能少；選出的資料則不能多，也不能少。
  • 對於明確要求排序的題目，你回答的 SQL 也必須明確寫出排序相關的指令，才能取得該題全部分數。
