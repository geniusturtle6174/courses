題目敘述與分數比率

以 MySQL 語言解答下列 LeetCode 問題，每題 25 分，最多採計至 100 ，且前兩題不同時採計。
  • 1683. Invalid Tweets。
  • 595. Big Countries。
  • 182. Duplicate Emails。
  • 1729. Find Followers Count。
  • 2356. Number of Unique Subjects Taught by Each Teacher。

注意事項

  • 本週有到課並解出至少一題，或者本週內完成正式請假者，可以在次週課堂 demo 未解出之題目並以八折計分，其餘狀況不收遲交。請假者須提供假條截圖，教師亦可能視情況要求你提出其他佐證，例如就醫證明等。
  • Demo 時，教師會要求你回答 SQL 語句的某個部分，是為了符合題目條件的哪個部分；或者題目條件的某個部分，撰寫在 SQL 語句中是哪個部分。
